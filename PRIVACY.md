# Privacy Policy — Black Hole Chrome Extension

**Last updated:** 2026-04-02

## Data Collection

Black Hole does **not** collect, store, transmit, or share any user data.

## What the extension does

- When activated by clicking the toolbar icon, the extension modifies the visual appearance of the current web page by making text transparent and creating particle animations
- All modifications happen locally in your browser tab
- When deactivated, all page modifications are fully restored

## Permissions

- **activeTab**: Used only to inject the visual effect into the currently active tab when you click the extension icon
- **scripting**: Used to inject the content script and stylesheet into the active tab

## No data leaves your browser

- No analytics or tracking
- No network requests
- No cookies or local storage
- No user data collection of any kind
